Key Patterns

Filter System

4 global filters shared via useFilters singleton composable: Time Period, Warehouse, Category, Order Status. Applied as query params to API calls.

Reactivity Model

Raw API data stored in Vue ref() values. Derived/computed data uses computed() properties for automatic re-rendering on filter changes.

Internationalization

EN/JA support via useI18n composable. Locale persisted to localStorage. Currency auto-switches between USD and JPY (rate: 1:150).
